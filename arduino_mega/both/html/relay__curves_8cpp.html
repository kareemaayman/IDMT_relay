Detailed Description

IDMT Protection Curve Implementations.

Implements IEC 60255-3 and IEEE C37.112 time-current characteristics (TCC).

IEC Formula: T = TMS * k / (M^α - 1) IEEE Formula: T = TDS * [A/(M^p - 1) + B]

Where M = I / I_pickup (current multiplier)

Calculate RMS voltage from ADC sample buffer.

Three-step process:

  1. Measure DC offset (average value - signals are biased to midpoint)
  2. Center each sample around zero
  3. Calculate RMS of centered (AC) signal

Works with any ADC bit width using Vref and ADC_res parameters. Example: Arduino Mega 2560 with 10-bit ADC (0-1023), Vref=5V, LSB≈4.88mV

Parameters
samplesPointer to array of ADC readings (uint16_t)
nNumber of samples in array
VrefADC reference voltage (e.g., 5.0V for Arduino)
ADC_resADC resolution max value (e.g., 1023 for 10-bit)
out_offset[out] Pointer to store measured DC offset in ADC counts. Pass NULL to skip storing offset.
Returns
RMS voltage in Volts
Return values
-1.0fon error (null pointer or invalid n)
