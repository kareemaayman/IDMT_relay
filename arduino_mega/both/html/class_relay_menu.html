Detailed Description

═══════════════════════════════════════════════════════════════════════════════ DISPLAY.H - Serial Menu Interface (115200 baud via USB/UART) ═══════════════════════════════════════════════════════════════════════════════

RelayMenu provides a text-based menu system for configuring relay parameters via Serial connection. Used primarily for initial setup and debugging.

Features:

  • Main menu with 6 options (Standard, Curve, TMS, Pickup, Inst M, Reset)
  • Numeric entry with floating-point support
  • Validation of input ranges
  • Real-time status display

Usage:

  1. Create instance with shared state: RelayMenu menu(shared_state);
  2. Call init() in setup() to display initial menu
  3. Call processInput(c) in loop() for each character from Serial

void RelayMenu::processInput ( char c)

Process a single character from Serial input Updates menu state and relay configuration based on user input

Parameters

Process a single character from Serial input This is the main state machine dispatcher that routes input to appropriate handlers

Current menu state determines how character is interpreted:

  • In MENU_MAIN: 1-5 to select what to change, 6 to reset
  • In other menus: either direct selection or numeric input

c: Character received from Serial (typically from Serial.read())
