\doxyref{LCDKeypad\+Menu}{p.}{class_l_c_d_keypad_menu} provides a hardware menu interface using a 4×3 matrix keypad and 20×4 I²C LCD display. This is the primary user interface for standalone operation (no PC connection needed).~\newline
 $\ast$ Keypad Layout:\+ 1 2 3 Keys 1-\/5 :\+ Configure relay settings 4 5 6 Key 7 :\+ Toggle menu ↔ protection mode 7 8 9 Key 9 :\+ View status
\begin{DoxyItemize}
\item 0 \# Key 0 :\+ Go back Key \# :\+ Reset latch or confirm entry Key $\ast$ :\+ Enter decimal point during numeric entry
\end{DoxyItemize}

Features:\+
\begin{DoxyItemize}
\item Real-\/time LCD display updates
\item Numeric entry with decimal support
\item Input validation with error messages
\item Protection mode status display during operation
\end{DoxyItemize}

Usage:\+
\begin{DoxyEnumerate}
\item Create instance:\+ \doxyref{LCDKeypad\+Menu}{p.}{class_l_c_d_keypad_menu} menu(lcd\+\_\+ref, shared\+\_\+state);~\newline
