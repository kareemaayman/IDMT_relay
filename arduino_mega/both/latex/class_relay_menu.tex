\doxyref{Relay\+Menu}{p.}{class_relay_menu} provides a text-\/based menu system for configuring relay parameters via Serial connection. Used primarily for initial setup and debugging.

Features:\+
\begin{DoxyItemize}
\item Main menu with 6 options (Standard, Curve, TMS, Pickup, Inst M, Reset)
\item Numeric entry with floating-\/point support
\item Validation of input ranges
\item Real-\/time status display
\end{DoxyItemize}

Usage:\+
\begin{DoxyEnumerate}
\item Create instance with shared state:\+ \doxyref{Relay\+Menu}{p.}{class_relay_menu} menu(shared\+\_\+state);
\item Call \doxyref{init()}{p.}{class_relay_menu_a45154e29ecc0f3a31495fff44afeac84} in setup() to display initial menu
\item Call process\+Input(c) in loop() for each character from Serial 
\end{DoxyEnumerate}
